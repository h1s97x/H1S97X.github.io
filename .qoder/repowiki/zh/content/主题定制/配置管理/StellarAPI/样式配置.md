# 样式配置

样式配置（`style`）控制主题的外观，包括主题色、字体、圆角、代码高亮等。

## 主题模式

### prefers_theme - 默认主题

```yaml
style:
  prefers_theme: auto  # auto / light / dark
```

**可选值**：
- `auto`：跟随系统（推荐）
- `light`：浅色模式
- `dark`：深色模式

## 字体配置

### font-size - 字号

```yaml
style:
  font-size:
    root: 16px
    body: 17px
    code: 85%
    codeblock: 0.8125rem
```

| 字段 | 说明 |
|------|------|
| root | 根字号，影响全局 |
| body | 正文字号 |
| code | 行内代码字号（相对） |
| codeblock | 代码块字号（绝对） |

### font-family - 字体

```yaml
style:
  font-family:
    body: 'system-ui, "Microsoft Yahei", "PingFang SC", sans-serif'
    code: 'Menlo, Monaco, "Cascadia Code", Consolas, monospace'
    codeblock: 'Menlo, Monaco, "Fira Code", Consolas, monospace'
```

## 颜色配置

### color - 主题色

```yaml
style:
  color:
    theme: 'hsl(192 98% 55%)'
    accent: 'hsl(14 100% 57%)'
    link: 'hsl(207 90% 54%)'
```

| 字段 | 说明 |
|------|------|
| theme | 主题色 |
| accent | 强调色 |
| link | 链接颜色 |

## 圆角配置

### border-radius - 圆角

```yaml
style:
  border-radius:
    card-l: 24px
    card: 16px
    card-s: 12px
    bar: 8px
    image-l: 24px
    image: 16px
    image-s: 8px
```

## 代码高亮

### codeblock - 代码块

```yaml
style:
  codeblock:
    scrollbar: 4px
    highlightjs_theme: https://gcore.jsdelivr.net/gh/highlightjs/cdn-release@11.9/build/styles/atom-one-dark.min.css
```

**常用主题**：
- `atom-one-dark` - Atom 深色
- `atom-one-light` - Atom 浅色
- `github` - GitHub 风格
- `monokai` - Monokai
- `vs2015` - Visual Studio 2015

## 侧边栏样式

### leftbar - 左侧栏

```yaml
style:
  leftbar:
    background-color-light: var(--card)
    background-color-dark: var(--card)
    background-image: url(...)
    blur-px: 100px
    blur-bg: var(--bg-a60)
    background-opacity: 0.8
```

## 完整配置示例

### 默认配置

```yaml
style:
  prefers_theme: auto
  font-size:
    root: 16px
    body: 17px
  color:
    theme: 'hsl(192 98% 55%)'
    accent: 'hsl(14 100% 57%)'
  border-radius:
    card: 16px
    image: 16px
```

### 极简配置

```yaml
style:
  prefers_theme: light
  font-size:
    root: 15px
  color:
    theme: '#000000'
  border-radius:
    card: 0px
    image: 0px
```
