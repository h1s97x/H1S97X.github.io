# 标签插件配置

标签插件（`tag_plugins`）提供了丰富的内容展示组件。

## note - 提示框

```yaml
tag_plugins:
  note:
    default_color: ''
    border: true
```

**使用方法**：

```markdown
{% note 这是一个提示框 %}
{% note warning 这是警告提示 %}
```

## checkbox - 复选框

```yaml
tag_plugins:
  checkbox:
    interactive: false
```

**使用方法**：

```markdown
{% checkbox 已完成的任务 checked %}
{% checkbox 未完成的任务 %}
```

## image - 图片

```yaml
tag_plugins:
  image:
    fancybox: false
```

**使用方法**：

```markdown
{% image /images/photo.jpg 图片描述 %}
```

## timeline - 时间线

```yaml
tag_plugins:
  timeline:
    max-height: 80vh
```

**使用方法**：

```markdown
{% timeline %}
{% timenode 2025-01-01 %}
事件描述
{% endtimenode %}
{% endtimeline %}
```

## gallery - 图库

```yaml
tag_plugins:
  gallery:
    layout: grid
    size: mix
    ratio: square
```

**使用方法**：

```markdown
{% gallery %}
![](/images/1.jpg)
![](/images/2.jpg)
{% endgallery %}
```

## 完整配置示例

```yaml
tag_plugins:
  note:
    default_color: ''
    border: true
  checkbox:
    interactive: false
  image:
    fancybox: true
  timeline:
    max-height: 80vh
  gallery:
    layout: grid
    size: mix
    ratio: square
```
