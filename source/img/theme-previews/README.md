# 主题预览图片

这个目录包含各个主题的预览图片，用于主题切换器界面。

## 图片规格
- 尺寸：建议 300x200 像素
- 格式：JPG 或 PNG
- 文件名：与主题名称对应

## 主题列表
- `stellar.jpg` - Stellar 主题预览
- `anzhiyu.jpg` - AnZhiYu 主题预览  
- `butterfly.jpg` - Butterfly 主题预览
- `icarus.jpg` - Icarus 主题预览
- `diaspora.jpg` - Diaspora 主题预览
- `default.jpg` - 默认预览图（当主题图片加载失败时使用）

## 添加新主题
1. 将主题预览图片放入此目录
2. 在 `theme-switcher.js` 中添加主题配置
3. 确保图片文件名与主题名称匹配
