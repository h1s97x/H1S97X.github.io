# 插件配置

Stellar 主题支持多种功能插件，按需启用。

## 图片插件

### fancybox - 图片灯箱

点击图片放大查看。

```yaml
plugins:
  fancybox:
    enable: true
    js: https://gcore.jsdelivr.net/npm/@fancyapps/ui@5.0/dist/fancybox/fancybox.umd.js
    css: https://gcore.jsdelivr.net/npm/@fancyapps/ui@5.0/dist/fancybox/fancybox.css
    selector: .md-text img:not([class])
```

## 数学公式

### katex - 数学公式渲染

```yaml
plugins:
  katex:
    enable: true
    inject: |
      <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.16.23/dist/katex.min.css">
```

**使用方法**：

```markdown
行内公式：$E=mc^2$

块级公式：
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

### mathjax - 数学公式渲染（备选）

```yaml
plugins:
  mathjax:
    enable: false
    js: https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.6/MathJax.js?config=TeX-AMS-MML_HTMLorMML
```

## 图表插件

### mermaid - 流程图

```yaml
plugins:
  mermaid:
    enable: true
    js: https://gcore.jsdelivr.net/npm/mermaid@v9/dist/mermaid.min.js
    theme: neutral
```

**使用方法**：

````markdown
```mermaid
graph LR
  A[开始] --> B[处理]
  B --> C[结束]
```
````

## 代码插件

### copycode - 代码复制

```yaml
plugins:
  copycode:
    enable: true
    default_text: 'Copy'
    success_text: 'Copied'
    toast: 复制成功
```

## 性能插件

### preload - 页面预加载

```yaml
plugins:
  preload:
    enable: true
    service: flying_pages
    flying_pages: https://gcore.jsdelivr.net/npm/flying-pages@2/flying-pages.min.js
```

## 轮播插件

### swiper - 图片轮播

```yaml
plugins:
  swiper:
    enable: true
    css: https://unpkg.com/swiper@10.3/swiper-bundle.min.css
    js: https://unpkg.com/swiper@10.3/swiper-bundle.min.js
```

## 排版插件

### heti - 中文排版

```yaml
plugins:
  heti:
    enable: false
    css: https://unpkg.com/heti@0.9/umd/heti.min.css
    js: https://unpkg.com/heti@0.9/umd/heti-addon.min.js
```

**注意**：此插件可能与代码块冲突，仅适用于纯中文博客。

## 最佳实践

### 1. 按需启用

只启用需要的插件，避免加载不必要的资源。

### 2. 使用 CDN

使用可靠的 CDN 加速资源加载。

### 3. 版本锁定

锁定插件版本，避免自动更新导致的兼容性问题。
