__ 4. 列出系统中所有登录用户：使用who命令。

Linux who命令用于显示系统中有哪些使用者正在上面，显示的资料包含了使用者 ID、使用的终端机、从哪边连上来的、上线时间、呆滞时间、CPU 使用量、动作等等。

· User Name

· Home directory

· Shell

· Login status

· mail status

· .forward

Linux mesg命令用于设置终端机的写入权限。

将mesg设置y时，其他用户可利用write指令将信息直接显示在您的屏幕上。

参数：

  • n 不允许其他用户将信息直接显示在你的屏幕上。

__ 8. 用长列表形式显示mydir目录的内容. 长列表列出的各栏分别显示了什么信息?文件myfile1和myfile2的大小是多少?

长列表的各栏显示的信息分别是文件的权限、硬链接数、所有者、所属组、文件大小、最后修改时间、文件名。
文件的大小显示在第5列。
__ 9. 返回到你的主目录然后使用 ls -R 命令来显示你的目录树. 解释显示出来的信息。

不可以，因为mydir目录不为空，rmdir只能删除空目录。
__ 12. 进入mydir目录删除该目录中的2个文件. 然后返回到你的主目录并且删除掉mydir目录.

__ 13. 用一条rm命令可否实现12题中的要求？如何实现？解释rm命令的作用。
是可以的，可以使用以下命令实现：
rm -r mydir
rm命令是用来删除文件和目录的，-r选项表示递归删除目录及其下的所有文件和子目录。

__ 19. 创建一个sub1目录, 在sub1中再创建一个 sub2 目录. 用一条命令来实现.

__ 23. 列出你的主目录的内容. 递归显示sub1目录中的所有文件和目录.

  1. 创建一个sub1目录，在sub1中再创建一个sub2目录，用一条命令来实现：
    mkdir -p sub1/sub2
  2. 转到sub2目录，创建一个文件myfile：
    cd sub1/sub2
    touch myfile
  3. 返回到你的主目录复制整个sub1目录树到tree1，分别递归显示目录sub和tree1里的所有文件：
    cd ~
    cp -R sub1 tree1
    ls -R sub1
    ls -R tree
  4. 现在你有了个目录树，sub1和tree1。把目录树tree1移动到sub1子目录中：
    mv tree1 sub1
  5. 列出你的主目录的内容，递归显示sub1目录中的所有文件和目录：
    ls -R ~
    ls -R sub1

请注意，上述命令假设当前目录是主目录（~）。如果不是，请先切换到主目录再执行命令。
